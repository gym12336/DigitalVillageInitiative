# 数乡计划视觉系统：数字田野博物馆

## 核心叙事

用数字博物馆的秩序整理乡土内容，用田野材料为数字界面提供温度。

- 数字展厅：地图、数据、档案编号、经纬度、采集状态；
- 田野档案：纸张色、宋体标题、陶土与铜金、口述与影像；
- 实践工具：保持信息清晰，不为装饰牺牲表单与任务效率。

## 色彩角色

| 角色 | 变量 | 用途 |
|---|---|---|
| 展厅黑 | `--museum-black` | 首页、展厅、地图背景 |
| 展厅墨绿 | `--museum-deep` | 深色区块与导航 |
| 档案纸 | `--paper` | 阅读区与内容页 |
| 玉石绿 | `--jade` | 主交互、状态、数据点位 |
| 铜金 | `--bronze` | 编号、展签、重要信息 |
| 陶土红 | `--clay` | 人文提示、分类与强调 |
| 数据微光 | `--data-glow` | 在线状态、扫描与地图反馈 |

青色微光只表示“数据正在运行”，不应大面积铺满页面。

## 字体角色

- UI、正文、表单：系统中文无衬线；
- 页面与村庄标题：系统宋体；
- 印章或极少量人文强调：系统楷体；
- 档案编号、坐标与状态：系统等宽字体。

不从不明来源下载字体，不通过公共 CDN 加载字体。确需分发开源字体时，必须保留其许可文件与版权声明。

## 形态与交互

- 内容卡片圆角控制在 0–8px；
- 用细线、编号和网格建立秩序，少用大阴影；
- 悬浮反馈以边框、局部颜色和轻微图像缩放为主；
- 不使用彩色 Emoji 作为核心导航图标；
- 遵守 `prefers-reduced-motion`，扫描线与滚动动画在减弱动画设置下停止。
