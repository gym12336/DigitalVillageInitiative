## 五、异常路径覆盖

每个补测文件至少覆盖 3 个异常路径类别：

| 类别 | 示例测试 |
|------|---------|
| **空/null/undefined 输入** | `summarize(null)` → `source:'empty'`；`extractFromText('')` → 不调 LLM |
| **网络/IO 失败** | `chatJSON` fetch reject → 重试一次；`getElevation` 双 API 均失败 → 返回 error |
| **超时** | `chatJSON` AbortController 20s 超时 → `DeepseekError`；`rateLimitedFetch` 重试耗尽 → 抛错 |
| **格式非法** | `chatJSON` 返回非 JSON → `DeepseekError`；`summarize` 返回非对象 → 兜底 |
| **并发竞态** | `TokenBucket` 50 并发 acquire 不超发；并发注册同用户名仅一个成功 |
