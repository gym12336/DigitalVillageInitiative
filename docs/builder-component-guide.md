## 快速入门

可视化搭建器采用**拖拽 + 属性编辑**的工作方式：

1. 从左侧**组件库**中拖拽组件到画布
2. 点击画布上的组件，在右侧**属性面板**中编辑内容
3. 画布支持拖拽移动组件位置、调整宽高

---

## 通用操作

### 画布操作

| 操作 | 方式 |
|------|------|
| 移动组件 | 拖拽组件到目标位置 |
| 调整大小 | 拖拽组件右下角手柄 |
| 选中组件 | 单击组件 |
| 删除组件 | 选中后点击属性面板标题栏的 🗑 按钮 |
| 画布缩放 | 鼠标滚轮 |
| 画布平移 | 拖拽空白区域 |

### 通用属性（所有组件）

所有组件共享以下属性，可在右侧面板编辑：

| 属性 | 说明 | 默认值 |
|------|------|--------|
| X | 组件左上角水平坐标 (px) | 拖放位置 |
| Y | 组件左上角垂直坐标 (px) | 拖放位置 |
| 宽 | 组件宽度 (px)，最小 20 | 各组件不同 |
| 高 | 组件高度 (px)，最小 20 | 各组件不同 |

## 组件类型

### 图表组件

图表组件是最常用的一类，通过 **CSV 文本** 驱动数据。在属性面板中可切换 7 种图表类型。

**默认尺寸：** 520 × 320 px

**通用图表属性：**

| 属性 | 说明 |
|------|------|
| 标题 | 图表顶部标题文字 |
| 图表类型 | 柱状图 / 堆叠柱状图 / 饼图 / 折线图 / 哑铃图 / 涨跌徽标 / 雷达图 |
| CSV 数据 | 逗号分隔的数据（含表头行），在属性面板中以等宽字体编辑 |

---

#### 1. 柱状图 `bar`

**用途：** 比较不同类别的数值大小。

**CSV 格式：**

```csv
label,value
类别A,35
类别B,68
类别C,42
类别D,55
类别E,27
```

**必填列：** `label`（类别名）、`value`（数值）

**视觉呈现：** 彩色垂直柱形，带数值标签、Y 轴刻度线和网格线。

---

#### 2. 堆叠柱状图 `stacked-bar`

**用途：** 展示多个系列在同一类别中的构成比例。

**CSV 格式：**

```csv
label,系列1,系列2,系列3
一月,10,20,15
二月,25,30,20
三月,35,28,22
```

**必填列：** `label`（类别名），其余列自动识别为数据系列（支持任意数量的系列）

**视觉呈现：** 垂直堆叠柱形，底部图例标注各系列颜色，柱顶显示合计值。

---

#### 3. 饼图 `pie`

**用途：** 展示各部分占整体的比例关系。

**CSV 格式：**

```csv
label,value
类别A,35
类别B,68
类别C,42
类别D,55
类别E,27
```

**必填列：** `label`（扇区名）、`value`（数值）

**视觉呈现：** 环形饼图（donut），扇区带标签和百分比，右侧图例列表。

---

#### 4. 折线图 `line`

**用途：** 展示数据随时间或顺序的变化趋势。

**CSV 格式：**

```csv
label,value
1月,35
2月,48
3月,42
4月,68
5月,55
6月,80
```

**必填列：** `label`（X 轴标签）、`value`（数值）

**视觉呈现：** 平滑曲线，数据点带圆点标记和数值标签，Y 轴刻度线和网格线。

---

#### 5. 哑铃图 `dumbbell`

**用途：** 对比"前—后"变化，展示各指标的起点与终点及变化幅度。

**CSV 格式：**

```csv
label,start,end
茶叶产量,120,210
农户年收入,8000,18500
合作社数量,3,12
村集体资产,50,320
```

**必填列：** `label`（指标名）、`start`（前值）、`end`（后值）

**视觉呈现：**
- 每行一条水平虚线连接起点（蓝色小圆）和终点（蓝绿色大圆）
- 两圆上方分别标注前值和后值
- 右侧显示变化百分比和方向箭头（▲ 上涨绿色 / ▼ 下跌红色）
- 每行独立比例尺，量纲差异大的数据也能清晰呈现

**示例数据解读：**

| 指标 | 前值 | 后值 | 变化 |
|------|------|------|------|
| 茶叶产量 | 120 | 210 | ▲ +75.0% |
| 农户年收入 | 8,000 | 18,500 | ▲ +131.3% |
| 合作社数量 | 3 | 12 | ▲ +300.0% |
| 村集体资产 | 50 | 320 | ▲ +540.0% |

---

#### 6. 涨跌徽标 `trend-badge`

**用途：** 以卡片形式展示关键指标的值及其涨跌趋势，适合做 KPI 看板。

**CSV 格式：**

```csv
label,value,change
销售额,128,560,+12.5%
用户数,42,091,+8.3%
转化率,3.28%,-0.5%
```

**必填列：** `label`（指标名）、`value`（当前值）

**可选列：** `change`（变化幅度文字，如 `+12.5%`）

**视觉呈现：**
- 卡片式网格布局（最多 3 列）
- 大数字展示当前值
- 底部彩色箭头标示涨跌（▲ 绿色 / ▼ 红色 / 灰色平）
- 单个指标时数字放大适配（适配强调场景）

---

#### 7. 雷达图 `radar`

**用途：** 多维度对比多个系列的指标表现，适合展示综合能力画像或均衡性分析。

**默认尺寸：** 520 × 380 px

**CSV 格式：**

```csv
label,产业兴旺,生态宜居,乡风文明,治理有效,生活富裕
李家村,80,65,72,88,70
全县平均,60,55,58,62,50
```

**必填列：** `label`（系列名），其余列自动识别为维度（至少需要 3 个维度）。支持任意多组数据叠加对比。

**视觉呈现：**
- 同心多边形虚线环（5 级刻度），第一条轴左侧标注刻度值
- 从中心偏移 15% 处向外辐射的维度轴线
- 各系列以半透明彩色多边形填充（`fill-opacity: 0.12`），顶点处带数据点圆标记
- 底部居中图例标注系列名称
- 维度标签沿多边形外圈排列，维度 ≥ 6 时自动缩小字号

### 文本组件

**用途：** 展示标题、段落、金句等文字内容。

**默认尺寸：** 300 × 96 px

**属性：**

| 属性 | 说明 | 可选值 | 默认值 |
|------|------|--------|--------|
| 文本内容 | 多行文本 | — | "新建文本" |
| 字号 | 字体大小 (px) | 8–200 | 34 |
| 粗细 | 字体粗细 | 常规(400) / 粗体(700) / 特粗(900) | 粗体(700) |
| 颜色 | 文字颜色 | 颜色选择器 | #1f2937 |
| 对齐 | 水平对齐 | 左对齐 / 居中 / 右对齐 | 左对齐 |
| 背景色 | 文本区域背景 | 颜色选择器 | 透明 |

---

### 图片组件

**用途：** 展示图片，支持远程 URL 和自动刷新（适合监控截图等场景）。

**默认尺寸：** 320 × 220 px

**属性：**

| 属性 | 说明 | 可选值 | 默认值 |
|------|------|--------|--------|
| 图片 URL | 远程图片地址 | — | （空） |
| 替代文本 | 图片加载失败时显示 | — | （空） |
| 填充模式 | 图片缩放方式 | Cover 裁剪 / Contain 完整 / Fill 拉伸 | Cover 裁剪 |
| 圆角 | 图片圆角 (px) | ≥ 0 | 0 |
| 自动刷新 | 是否定时重新加载 | 开/关 | 关 |
| 刷新间隔 | 自动刷新间隔 | ≥ 5 秒 | 60 秒 |

---

### 传感器组件

**用途：** 展示多个传感器（如农业物联网）的实时读数卡片组。

**默认尺寸：** 430 × 400 px

**属性：**

| 属性 | 说明 |
|------|------|
| 标题 | 传感器面板标题，默认"农业传感器监测" |
| 传感器列表 | 可动态增删的传感器项 |

**每个传感器包含：**

| 字段 | 说明 | 可选值 | 默认值 |
|------|------|--------|--------|
| 名称 | 传感器名称，如"温度" | — | （空） |
| 值 | 传感器读数 | — | 0 |
| 单位 | 度量单位，如"°C" | — | （空） |
| 状态 | 传感器状态 | 正常 / 注意(⚠) / 异常(🔴) | 正常 |

**操作：**
- 点击 **+ 添加** 新增一行传感器
- 点击 **×** 删除对应传感器

---

### 时间轴组件

**用途：** 展示按时间排序的事件里程碑，适合呈现项目推进、发展历程等时间序列叙事。

**默认尺寸：** 600 × 360 px

**属性：**

| 属性 | 说明 |
|------|------|
| 标题 | 时间轴顶部标题文字 |
| 事件列表 | 可动态增删的事件项 |

**每个事件包含：**

| 字段 | 说明 | 默认值 |
|------|------|--------|
| 日期 | 事件日期（如 `2020-03` 或 `2020年3月`） | （空） |
| 标题 | 事件名称 | （空） |
| 描述 | 事件补充描述（可选） | （空） |

**操作：**
- 点击 **+ 添加事件** 新增一个事件节点
- 点击 **×** 删除对应事件（至少保留 1 个）

**视觉呈现：**
- 水平时间线，一条细线（`#a5d7e4`）横贯组件中央偏上
- 事件节点等距分布，奇偶索引交替排列在时间线上下两侧
- 每个节点：彩色圆点（直径 14px）+ 竖线连接器 + 圆角卡片（日期标签 + 粗体标题 + 灰色描述，描述最多 2 行截断）
- 卡片底色为浅蓝灰（`rgba(44,125,160,0.03)`），节点颜色从调色板自动循环分配

### 数据表组件

**用途：** 以结构化表格展示行列数据，适合呈现荣誉列表、指标明细、对比数据等场景。

**默认尺寸：** 560 × 340 px

**属性：**

| 属性 | 说明 |
|------|------|
| 标题 | 表格顶部标题文字，默认"荣誉资质" |
| 列定义 | 可动态增删的列名列表 |
| 数据行 | 可动态增删的数据行，每行单元格数与列数对应 |

**操作：**
- 点击 **+ 添加列** 新增一列（所有已有行自动追加空单元格）
- 点击列名旁的 **×** 删除该列（同时删除所有行中对应列的数据）
- 点击 **+ 添加行** 新增一行（按当前列数填充空单元格）
- 点击行尾 **×** 删除该行（至少保留 1 行、1 列）

**视觉呈现：**
- 白色圆角卡片容器，标题栏与表体以细线分隔
- 表头行深色背景（`#245a73`），白色加粗文字、uppercase 字母间距
- 数据行奇偶交替底色（白 / 浅蓝灰）
- 单元格左对齐，文字溢出截断省略号
- 行数超出组件高度时表体内部垂直滚动，表头粘性固定

## 附录

### 组件库分类一览

| 分类 | 包含组件 |
|------|----------|
| 📊 讲「变化」— 帮扶对比 | 哑铃图、涨跌徽标 |
| 📈 讲「整体画像」— 概览 | KPI 卡组、雷达图 |
| ⏱️ 讲「过程」— 时间线 | 时间轴 |
| 🍩 讲「构成 / 分布」 | 饼图、堆叠柱 |
| 👥 讲「人与故事」 | 人物卡、金句块 |
| 🗺️ 讲「空间」— 地理分布 | 地图散点 |
| 🎬 封面开场 | 封面大图 |
| 🔢 单点强调 | 大数字 |
| 🔀 关系流向 | 桑基图 |
| 📅 时间频率 | 日历热力 |
| 🔄 交互对比 | 前后对比 |
| 🏆 荣誉佐证 | 数据表 |
| 🎬 媒体嵌入 | 视频嵌入 |

> **注意：** 部分组件（如桑基图、地图散点、日历热力、前后对比、视频嵌入等）目前为预留条目，尚未实现对应的专用渲染器，暂时默认渲染为柱状图。后续版本将逐步支持。

### CSV 数据编写规范

1. **第一行必须是表头**，包含列名（如 `label,value`）
2. **分隔符为英文逗号** `,`，不支持中文逗号
3. **表头名称必须精确匹配**：`label`、`value`、`start`、`end`、`change` 等
4. **数值列**支持整数和小数（如 `35`、`3.28`）
5. **空行会被自动忽略**
6. **缺少必填列**时图表将显示为"无数据"状态
